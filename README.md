# Healthcare IDS Diabetes Prediction System by Collins Agofure (MAYOR)

This project simulates an Intrusion Detection System (IDS) for healthcare by predicting diabetes using machine learning.

## Features
- Data preprocessing
- Multiple ML models
- Evaluation metrics
- Streamlit web application
- Visualization charts

## Run Instructions

### Install dependencies
pip install -r requirements.txt

### Train the model
python train_model.py

### Run the Streamlit app
streamlit run app.py
